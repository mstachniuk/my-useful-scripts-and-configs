Keyboard Layout for Mac.
It based on PL layout + makes German characters possible:
ü Option + [
ü Option + Shift + [
ö Option + ;
Ö Option + Shift + ;
ä Option + '
Ä Option + Shift + '
ß Option + \

Instalation
Just copy pl-de.bundle to /Library/Keyboard Layouts and chode in system preferences
sudo cp -Rv pl-de.bundle /Library/Keyboard\ Layouts
